# Continuous Scraping Project
Проект представляет собой автоматизированную систему для непрерывного скрапинга данных (на примере цен на книги), их сохранения, анализа и визуализации. Проект управляется оркестратором Prefect.
# Структура проекта
* ```flow.py``` - Точка входа для оркестратора Prefect. Содержит задачи (tasks) и пайплайн (flow).

* ```src/config.py``` - Настройки, ссылки и пути.

* ```src/scraper.py``` - Логика скрапинга (BeautifulSoup4 + requests). Точка входа для отдельного тестирования парсера.

* ```src/analyzer.py``` - Логика обработки сырых данных, подсчет статистики и генерация графиков (Pandas + Matplotlib).

* ```data/raw/``` - Здесь сохраняется сырая база данных ```scraped_data.csv```. Данные дозаписываются при каждом запуске.

* ```data/artifacts/``` - Здесь сохраняются результаты анализа: таблица ```summary_stats.csv``` и график ```price_trend.png```.

# Быстрый старт и запуск

1. Установка зависимостей:
```
python -m venv venv 
source venv/bin/activate  # Для Windows: venv\Scripts\activate
pip install -r requirements.txt
```

2. Как проверить скрапер отдельно (Точка входа скрапера):
Вы можете убедиться, что парсер работает без ошибок, запустив его напрямую:

```python -m src.scraper```


Ожидаемый результат: В консоли появится лог об успешном сохранении записей, а в папке ```data/raw/``` появится/обновится файл ```scraped_data.csv```.

3. Запуск полного цикла (Prefect Flow):

```python flow.py```


Это запустит сначала скрапер, а затем анализатор.

4. Где посмотреть артефакты:
После успешного запуска зайдите в директорию ```data/artifacts/```.

* Откройте ```price_trend.png```, чтобы увидеть график изменения цен.
* Откройте ```summary_stats.csv```, чтобы увидеть агрегированную таблицу.

5. Как запустить GUI Prefect (Dashboard):
В отдельном окне терминала выполните:

```prefect server start```


После этого откройте в браузере https://www.google.com/search?q=http://127.0.0.1:4200, чтобы увидеть дашборд с историей ваших запусков.
